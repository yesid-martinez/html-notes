HTML

HyperText Markup Language

Etiquetas < >

Las etiquetas HTML son fragmentos de código que nos permiten crear elementos HTML. Los elementos son la estructura básica de HTML. Dichos elementos tienen dos propiedades básicas: atributos y contenido.

Etiquetas (<head> </head>)

Etiqueta Atributos Contenido Función
<meta/> charset="UTF-8" X Permite mostrar caracteres especiales
<title> </title> X Titulo del documento Define el titulo del documento que se muestra en la ventana del browser
<link/> rel="icon" y href="" Link hacia la imagen Permite agregar una pequeña imágen junto al titulo del documento

Para obtener un mayor posicionamiento de la página utilizamos las siguientes etiquetas:

Etiqueta Atributos Contenido
<meta/> name="keywords" y content="" Palabras clave relacionadas con la página
<meta/> name="description" y content="" Descripción de la página
<meta/> name="author" y content="" Nombre autor
<meta/> name="robots" y content="" index / noindex

Etiquetas (<body> </body>)

Etiqueta Atributos Contenido Función
<header> </header> X Undefined Encabezado de la página
<nav> </nav> X Undefined Barra de navegación, se utiliza con listal "ul"
<article> </article> X Undefined Indica un articulo, se puede dividir en secciones
<section> </section> X Undefined Indica la sección de un articulo
<div> </div> X Undefined Definir un bloque de contenido o sección de la página
<span> </span> X Undefined Contenedor en línea. Permite aplicar estilo al texto o agrupar elementos en línea
<main> </main> X Undefined Representa el contenido principal de la página
<hr/> X X Representa un cambio de tema entre párrafos (línea horizontal)
<details> </details> X <summary> </summary> Es usada como un widget de revelación a través del cual el usuario puede obtener información adicional
<summary> </summary> X Titulo Define un texto visible para la etiqueta 'details'. La usamos con un párrafo hermano de esta etiqueta
<fieldset> </fieldset> X Inputs y/o <legend> </legend> Agrupa los elementos de un input
<legend> </legend> X Titulo Nombra un grupo creado con 'fieldset'. Añade un titulo aclaratorio sobre qué tipo de información se está agrupando en el recuadro
<aside> </aside> X Undefined Articulo secundario o contenido con menor relevancia. Representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento

Etiquetas de texto

Etiqueta Atributos Contenido Función
<p> </p> X Undefined Indica un párrafo
<b> </b> X Undefined Modificador de texto, negrita
<i> </i> X Undefined Modificador de texto, italica
<strike> </strike> X Undefined Modificador de texto, tachada
<small> </small> X Undefined Modificador de texto, pequeña
<h1> </h1> X Undefined Titulo del documento. Se recomienda un solo uso por página
<h2>, <h3> ... <h6> </h6> X Undefined Subtitulos del documento
<br/> X X Produce un salto de línea en el texto. Es útil en situaciones donde la división de las líneas es significante

Listas

Tenemos dos tipos de listas: Listas ordenadas y desordenadas.
Para crear una lista usamos las etiquetas <ul> </ul> (Unordered list) o <ol> </ol> (Ordered list)
Dentro de estas etiqueta usamos la etiqueta <li> </li> para cada elemento que ingresemos a la lista

Unordered list

  • Elemento 1
  • Elemento 2
  • Elemento 3

Ordered list

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Podemos modificar el estilo de las listas.

Forms / Input

El elemento 'form' representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web.

Usamos este elemento con 'form' y dentro del form usamos los inputs.

Un input es un ingreso de datos por parte del usuario, el cual nos permite recibir los datos y programar una acción con estos, usualmente lo usamos en formularios.

Usamos inputs con 'input type="tipo_de_input"'.

Para hacer un input obligatorio usamos 'required=""'

La etiqueta 'form' tiene el atributo 'action' que nos permite dar un valor para indicar información sobre ese input al backend

La etiqueta 'label' con el atributo 'for' dandole a este como valor el id del input nos permite darle un titulo a ese input, permitiendo que este también nos permita entrar al input.

Aquí algunos ejemplos de inputs:

Input checkbox

Input radio (Para seleccionar solo una opción debo darle el atributo name a los inputs para relacionarlos y con el atributo checked le damos un valor por defecto)

Input reset

Input Submit (Para enviar la información al servidor, permite la etiqueta 'value')

Input button (También sirve para enviar información al servidor)

Hay más tipos de inputs, pero estos son los más comunes.

Podemos poner un area para texto (Similar a un input type text) con la etiqueta 'textarea' y podemos darle los atributos 'cols' (Cantidad de caracteres que deseemos por fila) y 'rows' (Cantidad de filas que queremos en el textarea) para ajustar el tamaño del textarea.

Para quitar la opcion de resize usamos: resize: none

También tenemos la opción de hacer una lista de selección en el formulario mediante la etiqueta 'select' indicandole con la etiqueta 'option' la cantidad de opciones que deseemos.

Y como en los inputs con checked, podemos darle el atributo 'selected' para elegir un valor por defecto.

Enlaces

iframe

El elemento 'iframe' representa un contexto de navegación anidado, el cual permite incrustrar otra página HTML en la página actual (Nos permite ingresar contenido de otras páginas web y darle atributos con 'iframe src="link" iframe')

Por ejemplo, un video de youtube:

Para agregarlo usamos 'iframe src="https://www.youtube.com/embed/link_yt" iframe'

Podemos darle atributos como: 'frameborder="0/1"' (Para darle borde)

También tenemos los atributos predeterminados de youtube: 'allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen iframe'

Href

Este elemento nos permite referenciar un link en un texto o una línea de codigo para determinadas funciones. Lo usamos con: 'a href="link" a'

Por ejemplo, un link a mi instagram:

Instagram

Para que el link vaya hacia otra página usamos el atributo 'target="_blank"

Id

Con el atributo 'id' le damos un identificador único a algún elemento del codigo. Al usarlo damos como atributo al elemento: 'id="nombre"'

Para referir el id usamos 'a href="#nombre_id" a'

Para mencionarlo en css usamos '#nombre_id'

Class

Con el atributo 'class' podemos, al igual que el id, identificar algún elemento del codigo, con la diferencia de que este nos permite darle un mismo identificador a varios elementos a la vez.

Para mencionarlo en css usamos '.nombre_class'

No podemos referir una clase en un link como con el id

Image

Podemos hacer que al dar un click en un imagen esta tenga un link, esto lo hacemos anindando la etiqueta 'img' en la etiqueta 'a' que contiene el link

Multimedia

Podemos agregar imágenes locales (Dentro de la carpeta del archivo) o referir un link de la web con 'img src="link"'

De la misma manera podemos agregar un video local o de la web con 'video src="link_video"' (Al agregar videos de ruta local debemos agregar el atributo 'controls')

Algunos de los atributos que tienen estos elementos son:

  • 'alt="texto"' (Texto alternativo, se muestra cuando la imagen no se muestra.
  • 'title="texto"' (Al pasar el mouse por encima de la imagen/video se muestra el titulo)

    • También podemos agregar audios con 'audio src="audio_link"' con el atributo controls

Tablas

Para hacer una tabla usamos la etiqueta 'table'.

Dentro del 'table' con 'tr' definimos las filas verticales.

Dentro del 'tr' con 'td' definimos las filas horizontales.

También podemos darle propiedades a la tablas, para facilitar esto tenemos las etiquetas 'thead' y 'tbody' para el respectivo head y body de la tabla.
